Acerca de la Aplicación de Vacunación

Esta aplicación web fue desarrollada para proporcionar acceso fácil y rápido a los datos históricos de vacunación contra el sarampión en niños de 12-23 meses en Panamá. Los datos son almacenados y gestionados usando Redis, y la interfaz está construida con Flask y Jinja2.

Objetivos

  • Facilitar el acceso a los datos de vacunación.
  • Promover la transparencia en los datos de salud pública.
  • Apoyar la investigación y análisis de datos sobre la vacunación.

Tecnologías Utilizadas

Las tecnologías principales usadas en esta aplicación incluyen:

  • Flask: Un micro framework de Python para el desarrollo web.
  • Redis: Una base de datos en memoria que ofrece almacenamiento rápido de datos.
  • Jinja2: Un motor de plantillas para Python, utilizado para construir las vistas de la aplicación.
